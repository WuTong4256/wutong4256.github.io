vscode编辑器

vscode介绍

通俗来讲就是你编辑代码的一个工具

vscode安装

此处省略安装方法

vscode编辑器常用快捷键

快捷键 功能
Ctrl + c/v 复制/粘贴
Alt + shift 拖动鼠标 选中某块区域
Alt + ↑/↓ 向上/向下移动行
Shift + Alt + ↑/↓ 向上/向下复制行
Ctrl+ / 注释
Ctrl + +/- 放大界面/缩小界面
Ctrl+ Alt + ↑/↓ 添加多个光标（多行同时输入同样的内容）
Alt + z 切换是否换行
Ctrl + d 选定多个相同单词
Ctrl + h 替换/全局替换
Alt + shift+左箭头/右箭头 选中行
Ctrl + 左箭头/右箭头 光标定位到单词首/单词尾
Home/End 光标定位到首行/行末
Ctrl + shift+ 左箭头/右箭头 选中单词

vscode编辑器适用插件

插件名称 插件功能
Chinese (Simplified) Language Pack for Visual Studio Code 软件汉化
Auto Rename Tag 自动完成另一侧标签的同步修改
Bracket Pair Colorizer 给嵌套的各种括号加上不同的颜色
Path Intellisense 智能路径提示，可以在你输入文件路径时智能提示
JavaScript(ES6) code snippets ES6语法智能提示以及快速输入
Vetur Vue多功能集成插件
EsLint 语法纠错
